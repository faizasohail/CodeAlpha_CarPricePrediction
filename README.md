# Car Price Prediction using Machine Learning

## 🎯 Objective
Predict car selling price using machine learning regression models.

## 📁 Dataset Features
- Car Age
- Fuel Type
- Transmission
- Seller Type
- Mileage

## 🛠 Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Model Used
Linear Regression

## 📊 Result
Model predicts car prices based on input features with good accuracy.
